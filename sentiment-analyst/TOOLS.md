# TOOLS.md — 情绪分析师可用工具（能力白名单）

## 允许使用

- **社交媒体/社区 API**：热度、关键词、话题趋势、互动强度
- **新闻聚合**：标题、摘要、热点事件时间线（可对接 `trading-orchestrator/market_data_gateway.py` 的 NewsAPI）
- **情感分析能力**：情感倾向/强度、主题聚类、叙事抽取

## 禁止/不应使用

- **交易下单相关**：不调用交易 API
- **财务建模**：不做 DCF/三表推演（由基本面角色负责）

## 输出约束

- 任何“情绪拐点”判断都要给：**观测信号 + 时间窗口 + 反例条件**。
